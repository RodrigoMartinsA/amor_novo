4️⃣ COMPRIMIR IMAGENS (IMPORTANTE!)

Para o site rodar rápido, comprima as fotos:

❌ RUIM: Foto JPG com 5MB
✅ BOM: Foto JPG com 200-500KB

FERRAMENTAS GRÁTIS:
- TinyPNG.com (arraste e solta)
- Squoosh.app (do Google)
- ImageOptim (Mac) ou FileOptimizer (Windows)

DICAS:
- Máximo 2000x2000px de resolução
- Qualidade 80% geralmente é imperceptível
- Use JPG para fotos, PNG para gráficos

5️⃣ PERSONALIZAR CORES NO CSS

EDITE 'styles.css' PARA MUDAR CORES:

/* Fundo da página */
body {
  background-color: #fafafa; /* ← Mude aqui */
}

/* Texto principal */
body {
  color: #1a1a1a; /* ← Mude aqui */
}

/* Header/Navbar */
.header {
  background: rgba(255, 255, 255, 0.95); /* ← Branco */
}

ENCONTRE CORES BONITAS EM:
- coolors.co (gerador de paletas)
- colorhexa.com (conversor HEX)
- Pinterest (busque "color palette")

8️⃣ OTIMIZAÇÕES EXTRAS

✅ LAZY LOADING (já está incluído):
<img src="..." loading="lazy">

✅ SRCSET para diferentes resoluções:
<img 
  src="images/foto-500.jpg"
  srcset="images/foto-500.jpg 500w,
          images/foto-1000.jpg 1000w"
  alt="Descrição"
  loading="lazy"
>

✅ WEBP (formato mais rápido):
<picture>
  <source srcset="images/foto.webp" type="image/webp">
  <img src="images/foto.jpg" alt="Descrição">
</picture>

9️⃣ TESTAR LOCALMENTE

OPÇÃO 1 - Navegador direto (Simples):
1. Abra o arquivo index.html no navegador
   (Basta dar double-click no arquivo)

OPÇÃO 2 - Servidor Local (Recomendado):
Se você tem Python instalado:
  - Digite no terminal: python -m http.server
  - Abra: http://localhost:8000

Se você tem Node.js instalado:
  - Digite no terminal: npx serve
  - Siga as instruções na tela

OPÇÃO 3 - VS Code:
1. Instale a extensão "Live Server"
2. Clique direito em index.html
3. Clique em "Open with Live Server"

💬 COMANDOS ÚTEIS NO TERMINAL

Comprimir fotos em bulk (ImageMagick):
mogrify -quality 80 -resize 2000x2000 images/*.jpg

Converter para WebP:
cwebp images/foto.jpg -o images/foto.webp

Verificar tamanho:
ls -lh images/

Servidor Python simples:
python -m http.server 8000

Servidor Node:
npx http-server
